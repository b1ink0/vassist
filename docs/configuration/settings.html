Settings Overview

VAssist provides extensive customization options through the Settings Panel. Access settings by clicking the gear icon in the chat interface.

Settings Categories​

UI Settings​

Configure companion appearance and positioning:

  • Enable/Disable Companion - Toggle virtual companion on/off
  • Portrait Mode - Show upper body only (vs full body)
  • Position Presets - Choose where companion appears on screen
  • FPS Limit - Control rendering performance
  • Auto-Load - Load companion on all pages automatically
  • AI Toolbar - Enable/disable toolbar on text selection and input focus
  • Colored Icons - Use colored icons instead of monochrome
  • Background Detection - Adaptive theme based on page background
  • Keyboard Shortcuts - Configure shortcuts for quick access

Learn more about UI Settings →

LLM Settings​

Configure AI language model providers:

  • Chrome AI - Built-in local AI (recommended)
  • Fallback Providers - OpenAI compatible APIs when Chrome AI unavailable
  • System Prompts - Customize AI behavior and personality
  • Model Configuration - Temperature, max tokens, etc.

Learn more about LLM Settings →

TTS Settings​

Configure Text-to-Speech for voice responses:

  • Kokoro TTS - On-device neural TTS (recommended)
  • Voice Selection - Choose voice characteristics
  • Speed & Pitch - Adjust speech parameters
  • Fallback Providers - OpenAI TTS or custom OpenAI-compatible endpoints

Learn more about TTS Settings →

STT Settings​

Configure Speech-to-Text for voice input:

  • Chrome AI STT - Built-in speech recognition
  • Language Selection - Choose recognition language
  • Fallback Providers - OpenAI Whisper or custom OpenAI-compatible endpoints

Learn more about STT Settings →

AI Features​

Enable or disable individual AI capabilities:

  • Summarizer - Text summarization
  • Translator - Language translation
  • Rewriter - Text rewriting and improvement
  • Writer - Content generation

Learn more about AI Features →

Accessing Settings​

From Chat Interface​

  1. Open chat window
  2. Click gear icon in controls panel at bottom
  3. Navigate between setting categories using tabs
