# Keyboard Shortcuts

Configure custom keyboard shortcuts to quickly access VAssist features without using the mouse.

## Enable Keyboard Shortcuts

**Default**: Disabled

Toggle to enable or disable all keyboard shortcuts:

- **Enabled** - Shortcuts are active and will trigger actions
- **Disabled** - Shortcuts are inactive (default)

When disabled, shortcut bindings are still saved but won't trigger any actions.

## Available Shortcuts

VAssist supports two customizable keyboard shortcuts:

## Recording Shortcuts

### How to Set a Shortcut

1. **Click the input field** - Shows "Press keys..." placeholder
2. **Press your key combination** - Use modifiers + a key
3. **Shortcut is saved** - Displays formatted shortcut (e.g., `Ctrl+Shift+C`)
4. **Click X button to clear** - Removes the shortcut

### Requirements

:::warning Modifier Required
At least **one modifier key** must be used to avoid conflicts with normal typing:
- **Ctrl** (Control)
- **Alt**
- **Shift**
- **Meta** (Windows key / Cmd key on Mac)
:::
