# Getting Started

Welcome to VAssist documentation! VAssist is a Chrome extension that brings an AI-powered virtual companion to your browser.

![VAssist Overview](/img/overview.gif)

## What is VAssist?

VAssist has three main components that work together to enhance your browsing experience:

- **AI Toolbar** - Appears when selecting text or focusing inputs. Provides summarize, translate, rewrite, grammar fix, tone change, image analysis, and voice dictation
- **Chat Interface** - Chat with AI about current page or anything else. Supports image/audio attachments, voice mode, conversation history with branching
- **Virtual Companion** - Animated character that appears on pages. Shows different states (idle, thinking, speaking) with lip-synced speech

## Key Features

- **Local Processing** - All AI runs on-device using Chrome's built-in AI
- **No API Keys** - No subscriptions, no external services needed
- **Privacy First** - Your data stays on your device
- **Works Offline** - After initial model download

## Requirements

- Chrome Browser version 138 or newer
- Chrome AI flags enabled
