# AI Toolbar

The AI Toolbar appears when you select text on any website or focus on input fields. It provides context-specific AI tools right at your fingertips.

![AI Toolbar](/img/toolbars.png)

## Text Selection Tools

### üìù Summarize

Turn walls of text into digestible chunks. Uses Chrome's Summarizer API to give you:

- **Headline** - One punchy line summarizing the main point
- **Key Points** - Bullet list of the most important ideas
- **Teaser** - Short preview that hooks you in

Perfect for quickly understanding long articles, documentation, or any text-heavy content.

### üåç Translate

Speak the language, any language. Powered by Chrome's Translator API and Language Detector API:

- Translate to 100+ languages instantly
- Helps to detect the language of selected text
- Works completely offline once language packs are downloaded
- Maintains formatting and context

Great for reading foreign language websites, international emails, or learning new languages.

### üìö Dictionary

Your personal word expert. Uses Chrome's AI:

- Get definitions on the spot
- Find synonyms and antonyms
- Learn correct pronunciation
- See real-world usage examples

No need to switch tabs or open separate apps - instant vocabulary help wherever you're reading.

### ‚úçÔ∏è Rewrite

Make your words shine. Chrome's Rewriter API helps you:

- **Fix Spelling & Grammar** - Clean up typos and grammatical mistakes
- **Change Tone** - Go formal, casual, or professional based on context
- **Adjust Length** - Make it shorter, expand it, or keep it concise
- **Improve Clarity** - Simplify complex sentences for better understanding
- **Custom Rewrites** - Tell it exactly what you want and it adapts

Appears in text input fields on text selection to help you refine what you write.

- Create content based on your ideas

- Perfect for brainstorming and drafting

- Real-time transcription as you speak

### üñºÔ∏è Image Analysis

Just hover over any image on a webpage to access powerful vision AI:

- **Describe** - AI tells you what's in the image using Multimodal Prompt API
